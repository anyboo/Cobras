    	导出类：CrypDll
    
	两个正式方法：
	bool codeRegOK(const string &);	// 注册激活
	string getApllyCode();		// 取机器码
	
	两个测试方法：
	string getMac();       //取MAC地址（测试用，发布时取消）
	string getRCode();     //取注册码（测试用，发布时取消）


	codeReg流程：
	1、取本地网卡MAC地址生成机器码（以前叫申请码），根据机器码生成注册激活码；

	2、根据用户输入的注册激活码进行下列判断；

	3、如果和系统生成的注册激活码一致则注册成功（同时写本地AuthorFile文件，记录注册激活码），注册结束，返回True。

	4、不对则读本机上的AuthorFile文件，提取注册激活码匹配，成功则显示文件注册成功，注册结束，返回True。

	5、本地无注册文件或注册文件里面的注册激活码和计算值出来的注册激活码不一致则注册失败，注册结束，返回False。
